Добро пожаловать, на этом сайте вы можете:
Сначала авторизоваться используя Github или Google.
Затем добавить все каналы youtube которые вы ходите СЛУШАТЬ в свой список каналов. Для этого просто добавляйте URL со ссылкой на канал на соответствующей странице.
Для каждого из добавленных вами каналов будет создан RSS поток со ссылками на эпизоды канала в аудио-формате. Этот список будет обновляться в соответствии с обновлениями основного канала. Сайт сам будет скачивать видео эпизоды и преобразовывать файлы в аудио формат. Вам лишь необходимо подписаться на соответствующий RSS для канала и слушать в своё удовольствие.
Ссылку на RSS канал для подписки можно взять в поле RSS feed на странице список каналов для каждого канала. Кроме этого, если вы ходите получить RSS поток со всеми эпизодами всех ваших каналов, просто подпишитесь на специальный RSS RSS feed "все файлы" и получайте все свои файлы в одном потоке.
